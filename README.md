# Analytics---project
Analytics system for Campus-Kul
his project builds an analytics system to track growth, activation, retention, engagement, and jobs.


#Data Sources
- users.csv  
- events.csv  
- posts.csv  
- jobs.csv
- notice.csv
- application.csv
- news.csv

 #How to run
 -Download files
 -open jupyter notebook , Run `file_name.ipynb', Generate report .

 #KPI Definitions
 - DAU = Daily Active Users
 - WAU -Weekly Active Users
 - MAU-Monthly Active Users. 
- Activation Rate = Completed signup / Total users.  
- Retention = how many user return back.
- Engagement - how active user are.
